🏙️ GeoML Urban Sprawl Detector

Satellite image segmentation + AI-generated policy reports.

How it works

  1. Upload a satellite image (e.g. from Esri, Google Maps)
  2. DeepLabV3 ResNet50 (PyTorch) segments buildings (COCO class 8)
  3. Pixel count → Zephyr-7B via Hugging Face Inference API → policy memo

Run Locally

git clone https://github.com/Klurien/GeoML-Urban-Sprawl
cd GeoML-Urban-Sprawl
python -m venv venv && source venv/bin/activate
pip install -r requirements.txt
echo "HF_TOKEN=your_hf_token" > .env
python app.py

Open http://localhost:7860
